# Git学习

##基本操作

### git init——初始化仓库

git init  初始化当前目录为git仓库

### git status——查看仓库的状态

git status

### git add——向暂存区中添加文件

git add <FileName>  将<FileName>添加到暂存区

### git commit——保存仓库的历史记录

git commit -m "message"  记述一行提交信息

git commit 记述详细提交信息

### git log——查看提交日志

git log    查看commit历史

### git diff——查看更改前后的差别

git diff    查看当前工作树与暂存区的差别

git diff HEAD    查看工作树和最新提交的差别
